# Student Depression Analysis Dashboard

## Overview
This Tableau dashboard analyzes factors affecting student depression.

## Features
- Depression Overview
- Gender Analysis
- Age Group Analysis
- Sleep Impact
- Academic Pressure Impact
- Financial Stress Impact
- Study Satisfaction Analysis

## Tools Used
- Tableau Public
- Excel / CSV
- GitHub
